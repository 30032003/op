Security Concepts Assignment 1 - Nmap Lab Notes (Exam Ready)

1. Install Nmap

-   Ubuntu:

    sudo apt update
    sudo apt install nmap -y

-   Check version:

    nmap --version

2. Basic Scan

    nmap <target>

Example:

    nmap 15.207.152.55

3. SYN Scan (Stealth Scan)

    sudo nmap -sS <target>

4. Service Version Detection

    sudo nmap -sV <target>

5. OS Detection

    sudo nmap -O <target>

6. Save Report

    sudo nmap -sS -sV -O -oX report.xml <target>

7. Convert to HTML (optional)

    xsltproc report.xml -o report.html

8. Understand Report

-   Open = Service available
-   Closed = Port reachable but no service
-   Filtered = Firewall/ACL blocking scan
-   Product = Running software
-   Version = Software version
-   OS Detection = Estimated operating system

13. Viva Questions

Q. What is Nmap? Ans: Network Mapper used for network discovery and
security auditing.

Q. Difference between Open and Filtered? Ans: Open means service is
available. Filtered means firewall blocks probes.

Q. What does -sS do? Ans: TCP SYN (half-open) scan.

Q. What does -sV do? Ans: Detects service versions.

Q. What does -O do? Ans: Detects operating system.

Q. Why save report? Ans: Documentation and analysis.
